# NetGuard Pro - Backend API

Backend for NetGuard Pro Flutter app.

## APIs:
- `/api/network/info` - Network info
- `/api/scan/ports` - Port scanner
- `/api/analyze/password` - Password analyzer
- `/api/cve/lookup` - CVE database
- `/api/router/detect` - Router detector
- `/api/wps/calculate` - WPS PIN calculator
- `/api/devices/scan` - Network devices scanner
- `/api/wifi/signal` - WiFi signal analyzer
- `/api/report/generate` - Security report generator
- `/api/ai/recommend` - AI-powered security recommendations
